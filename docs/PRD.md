# BizSim — Product Requirements Document

**Version**: 0.1 (MVP)
**Date**: 2026-03-28
**Author**: 전유성 (luma)

---

## 1. Overview

BizSim은 Markstrat에서 영감을 받은 AI 기반 한국어 경영 시뮬레이션 게임이다. 사용자가 한 개 분기의 경영 의사결정을 내리고, AI가 구현하는 소비자 페르소나와 인터뷰하며, 결과 및 재무제표를 확인한다. 경영학/마케팅 수업의 교육 도구로 설계되었다.

---

## 2. Target Users

| 페르소나 | 설명 | 핵심 니즈 |
|---|---|---|
| **교수 / 강사** | 경영/마케팅 수업 운영, 교보재 필요 | 수업에 즉시 투입 가능한 데모, 이론과의 연결 |
| **MBA/경영학 학생** | 의사결정 실습, 전략 개념 이해 | 직관적인 피드백, 학습 효과 |

---

## 3. Problem Statement

기존 Markstrat은:
- 고가 라이선스 ($50~100/student)
- 영어 전용, 한국 수업 맥락 부적합
- AI 인터랙션 없음 (ChatGPT 연동은 최근에야 추가)
- 설치 복잡, UX 노후화

BizSim은 **무료 + 한국어 + AI 소비자 인터뷰** 로 차별화한다.

---

## 4. Core User Stories

### 4.1 의사결정 (Decision)
- `US-01` 사용자는 6개 레버(가격, R&D, 광고, 생산량, 채널, 품질)를 슬라이더로 조정할 수 있다
- `US-02` 각 레버에 현재 값과 범위가 명확히 표시된다
- `US-03` 결정을 완료하면 시뮬레이션을 실행할 수 있다

### 4.2 AI 소비자 인터뷰 (Interview)
- `US-04` 사용자는 3개 소비자 페르소나 중 하나를 선택해 채팅할 수 있다
- `US-05` AI 페르소나는 사용자의 결정(가격, 품질 등)을 인지한 상태로 반응한다
- `US-06` 페르소나별 대화 기록은 세션 내 유지된다
- `US-07` 응답은 스트리밍으로 표시된다

### 4.3 결과 확인 (Results)
- `US-08` 시뮬레이션 결과: 판매량, 시장점유율, 수익, 고객만족도가 표시된다
- `US-09` 세그먼트별 수요가 바 차트로 시각화된다
- `US-10` AI 디브리프가 결과를 해석해준다

### 4.4 재무제표 (Financials)
- `US-11` P&L, 재무상태표, 현금흐름표를 탭으로 확인할 수 있다
- `US-12` 각 항목은 한국어 레이블과 금액 포맷(₩)으로 표시된다

## 5. Feature Scope

### MVP (현재)
- [x] 6개 의사결정 레버
- [x] 수요 모델 (가격/품질/광고/채널 탄성)
- [x] 3개 AI 소비자 페르소나 인터뷰 (스트리밍)
- [x] 시뮬레이션 결과 대시보드
- [x] 3종 재무제표 자동 생성

### v1.0 (다음)
- [ ] Gemini Flash API (비용 0원)
- [ ] UI 개선 (Markstrat 수준 대시보드)
- [ ] 의사결정 화면 좌우 레이아웃 (레버 + 미리보기)

### v2.0 (보류)
- [ ] 멀티 라운드 (분기 진행)
- [ ] 경쟁사 AI 의사결정
- [ ] 팀 멀티플레이어

## 6. Non-Goals (MVP)

- 멀티 유저/팀 기능
- 회원가입/저장/이력 관리
- 모바일 최적화
- 결제/라이선스

---

## 7. Success Metrics

| 지표 | 목표 |
|---|---|
| 교수 데모 피드백 | "수업에 써볼 수 있겠다" |
| 의사결정 → 결과까지 소요 시간 | < 3분 |
| AI 인터뷰 응답 지연 | < 2초 첫 토큰 |
| 재무제표 정확도 | 유닛 테스트 100% 통과 |

## 8. Constraints

- 팀: 1인 개발
